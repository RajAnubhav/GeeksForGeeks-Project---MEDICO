## Hospital Management System in Laravel 8, MySQL and Bootstrap 5
------------------

# Technologies
- Laravel 8
- MySQL
- Bootstrap 5
- Jquery 3
- Stripe Payment Gateway

## Panels
-----------------
1. Website -> patients
2. Admin Panel -> doctors

# Website
- Home Page
- Gallery Page
- Services
- About Us Page
- Testimonials
    - Customer can add testimonial after login
    - Only show when admin approve
- Booking Page
- Contact Page
- FAQ

# Admin Panel
- Login
- Forget Password
- Manage Website content
    - Home Page content
    - Gallery
    - About us content
    - Contact Enquiry
    - Testimonials
    - Services
    - FAQ Content
- Manage Room Categories
- Manage Rooms
- Manage Bookings
- Manage Staff
- Manage Salary of Staff
- Manage Patients
- Manage Payments of Customers
